Research Overview

This interface follows the same calm, structured direction as your reference site: clear problem framing, explainable machine learning, and a presentation-friendly narrative flow.

The Problem

Many organizations still respond to phishing, ransomware, and data breaches only after damage has already occurred. Localized risk models for Sri Lankan organizations are limited, especially those that combine both technical and awareness-related indicators.

Our Solution

This project provides a structured web application that accepts operational security inputs, classifies them into a risk level with a Random Forest model, and returns practical next-step recommendations through an explainable interface.

Security Controls

Measures firewall posture, MFA usage, encryption coverage, password policy strength, backup frequency, and monitoring readiness.

Human Awareness

Captures employee training, phishing test performance, and incident history to reflect the human side of cyber risk.

Explainable ML

Uses a Random Forest classifier to generate a traceable low, medium, or high risk output with practical recommendations.
